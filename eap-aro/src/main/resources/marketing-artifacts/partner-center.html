Description

JBoss EAP is a leading Java application server running some of the most mission-critical enterprise applications in the world. This solution automates most boilerplate steps to provision JBoss EAP on an Azure Red Hat OpenShift (ARO) cluster. Once initial provisioning is complete, you are completely free to customize deployments further. The solution is jointly developed by Red Hat and Microsoft.

The following resources are automatically provisioned by the offer.

  • ARO cluster (alternatively, you can deploy to an existing ARO cluster)
  • JBoss EAP Operator
  • Sample getting started application
  • Custom application deployment using Source-to-Image (S2I), based on JDK 11 and JBoss EAP 8
  • Virtual network and subnet

This offer requires a JBoss EAP subscription and works on a Bring-Your-Own-Subscription (BYOS) basis. Red Hat and Microsoft also provide basic step-by-step instructions on getting started with JBoss EAP and ARO without automated provisioning.

Red Hat and Microsoft provide similar solutions targeting Azure Virtual Machines and App Service. These options are linked in the Learn more section below.

You can reach out to the engineering team developing these offers by clicking the CONTACT ME button on the marketplace JBoss EAP on Azure overview page. Program managers, architects and engineers will get in touch and can assist you for free with your Azure migration.
