Description

This offer provisions the JBoss EAP operator on Azure Red Hat OpenShift and any necessary supporting Azure resources.

  • Network resources
    • An virtual network and a subnet.
  • Computing resources
    • An Azure Red Hat OpenShift.
  • Managed identity(to be removed)
    • An user assigned managed identity for running deployment scripts on created ARO.
  • Software components
    • The latest version of JBoss EAP Operator.
    • Source-to-Image application deployment if users choose to enable. The deployment environment is based on JDK 11 and EAP7.4.

Red Hat JBoss Enterprise Application Platform (JBoss EAP) is a leading Java application server running some of the most mission-critical enterprise applications in the world.

This offer automatically provisions several Azure resources to quickly create a JBoss EAP deployment on Azure Red Hat OpenShift. The automatically provisioned resources include virtual network, Azure Red Hat OpenShift cluster, network security group, and JBoss EAP Operator for OpenShift. The offer supports the latest versions of JBoss EAP 7, OpenJDK 8, and RHEL 8.

This offer requires a JBoss EAP subscription and works on a Bring-Your-Own-Subscription (BYOS) basis.

Description

JBoss EAP is a leading Java application server running some of the most mission-critical enterprise applications in the world. This solution automates most boilerplate steps to provision JBoss EAP on an Azure Red Hat OpenShift (ARO) cluster. Once initial provisioning is complete, you are completely free to customize deployments further. The solution is jointly developed by Red Hat and Microsoft.

The following resources are automatically provisioned by the offer.

  • ARO cluster (alternatively, you can deploy to an existing ARO cluster)
  • JBoss EAP Operator
  • Sample getting started application
  • Custom application deployment using Source-to-Image (S2I), based on JDK 11 and JBoss EAP 7.4
  • Virtual network and subnet

This offer requires a JBoss EAP subscription and works on a Bring-Your-Own-Subscription (BYOS) basis. Red Hat and Microsoft also provide basic step-by-step instructions on getting started with JBoss EAP and ARO without automated provisioning.

Red Hat and Microsoft provide similar solutions targeting Azure Virtual Machines and App Service. These options are linked in the Learn more section below.

You can reach out to the engineering team developing these offers by clicking the CONTACT ME button on the marketplace JBoss EAP on Azure overview page. Program managers, architects and engineers will get in touch and can assist you for free with your Azure migration.
